# Quartz v4

> “[那些] 敞开大门工作的人会遭到各种打扰，但 [他们] 也偶尔会得到一些关于这个世界是什么以及什么可能很重要的线索。” 
>
> — Richard Hamming

Quartz 是一套工具，帮助你免费将你的数字花园和笔记发布为网站。

Quartz v4 版本进行了程序重构，专注于用户的可扩展性和易用性。
